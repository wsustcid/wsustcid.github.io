博客搭建教程

https://zhuanlan.zhihu.com/p/102592286

1. 准备工作

  1. 注册github账号; 安装git; 本地生成ssh key 添加至github账户

  2. 安装node.js https://nodejs.org/en/

  3. 创建仓库 仓库名应该为：用户名.github.io 这个用户名使用你的GitHub帐号名称代替，这是固定写法; settings 页面底端出现 github pages Your GitHub Pages site is currently being built from the main branch.说明建立成功 (必须公开项目)

2. 正式搭建

  1. 新建文件夹Blog ，进入该文件夹，之后的操作都是在该文件夹下

  2. npm install -g hexo-cli ; 创建软连接ln -s /usr/tools/node/v13.6.0/bin/hexo /usr/local/bin/ 然后hexo -v 查看是否安装成功；

  3. hexo init 初始化博客

  4. hexo g 静态部署

  5. hexo s 查看部署成功的网页; http://localhost:4000 本地查看

  6. 将Hexo部署到github: 打开 _config.yml 文件 在文件底部添加

  7. 回到 Blog 文件夹中，打开 Git Bash，安装Git部署插件，输入命令 npm install hexo-deployer-git –save

  8. 然后分别输入以下三条命令：

  9. 完成以后，打开浏览器，输入 https://xxx.github.io 就可以打开你的网页了https://wsustcid.github.io/ws-garage.github.io/

  10. 现在虽然可以访问我们的网站，但是网址是GitHub提供的：http://xxxx.github.io 而我们想使用我们自己的个性化域名，这就需要绑定我们自己的域名。下一篇介绍解析域名和绑定域名。

将My_Blog初始化为git项目，上传至github; 然后尝试从另一台电脑克隆本项目，看能否继续写博客

3. 解析域名

  1. 购买域名 https://wanwang.aliyun.com/； https://dnspod.cloud.tencent.com/
  2. 解析域名 https://zhuanlan.zhihu.com/p/103813944

4. 个性化定制

  1. 进入博客根目录文件夹，下载next主题至theme文件夹下: git clone https://github.com/theme-next/hexo-theme-next.git 下载后 删除next下的.git文件夹

  2. ```
    打开根目录下的_config.yml(称为站点配置文件)，修改主题（注意冒号后都要有空格）：

    Site

    title: 枫叶苑 #标题
    subtitle: ‘’
    description: 选择有时候比努力更重要 #简介或者格言
    keywords:
    author: 枫叶 #作者
    language: zh-CN #主题语言
    timezone: Asia/Shanghai #中国的时区

    Extensions

    Plugins: https://hexo.io/plugins/

    Themes: https://hexo.io/themes/

    theme: hexo-theme-next #主题改为next

  3. 回到Blog根目录打开Git Bash，输入如下三条命令：

    hexo clean
    hexo g
    hexo d
